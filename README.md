# AI IT-Support Assistant 🤖

Мини-дашборд с AI-ассистентом для внутренней IT-поддержки. 

### Основной функционал:
- **Специализация:** Бот настроен строго на помощь в IT-вопросах.
- **История чатов:** Поддержка нескольких сессий, поиск по чатам и сохранение истории в локальный JSON.
- **Интерфейс:** Современная темная тема с градиентами, адаптивная верстка на Streamlit.
- **UX-фишки:** Стриминг ответов (печать в реальном времени), кнопка копирования ответа, создание новых чатов.
- **Безопасность:** Обработка ошибок API и защита от падения сервиса (Crashed-flow).

### Технологический стек:
- Python, Streamlit
- OpenAI SDK (интеграция через OpenRouter)
- Модель: DeepSeek V4 Flash

### Как запустить локально:
1. Клонируйте репозиторий: `git clone <ссылка>`
2. Установите зависимости: `pip install -r requirements.txt`
3. Создайте файл `.env` на основе `.env.example` и вставьте свой `OPENROUTER_API_KEY`.
4. Запустите приложение: `streamlit run app.py`
